​

. 이후에 A나 B를 넣어야 하니

스택에 값이 존재시

스택사이즈가 4의 배수이면 스택을 pop을 하면서 스트링 빌더에 A를 넣고

4의 배수가 아니면 2만큼을 스택에서 팝하며 B를 넣습니다.

남은 값은 한번더 4의 배수인지 점검하고 아니면 2만큼을 스택에서 팝하며 B를 넣는 걸 반복하여 A나 B를 넣으면 답안이 나옵니다.
